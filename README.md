# SatDishPointer

GEO卫星天线指向计算工具，用于计算地面站对准地球同步轨道卫星所需的方位角、仰角和极化角。

**Code written by LLM, directed by humans**

## 功能

- 计算方位角、仰角、极化角和站星距离
- 支持偏馈天线和正馈天线
- 支持自定义卫星经度输入（如 105.5E、75W）
- 支持 GPS 定位获取当前位置
- 中英文双语界面
- URL 参数分享功能

## 使用

### 在线访问

直接打开 `build/index.html` 或部署到任意静态服务器。

### 本地开发

```bash
npm install
npm start
```

### 构建

```bash
npm run build
```

构建产物在 `build/` 目录，可直接部署到静态服务器或本地打开。

## 计算原理

基于地球同步轨道卫星的几何关系：

- 地球半径：6371 km
- 同步轨道高度：35786 km
- 方位角：基于地面站纬度和经度差计算
- 仰角：基于地心角和轨道半径计算
- 极化角：基于经度差和纬度计算
